# Support

## Where to Ask What

Use GitHub Issues for:

- reproducible bugs
- broken docs
- clearly scoped feature requests

Use GitHub Discussions for:

- architecture questions
- setup questions
- design tradeoffs
- evaluation ideas
- showcase and feedback threads

Use the README and docs first for:

- quickstart
- local operations
- demo mode
- screenshots and release context

## What Makes Support Faster

Include:

- your goal
- what command or endpoint you ran
- the exact error
- relevant logs
- OS and runtime details when relevant

For UI problems, include:

- browser
- route
- request that failed
- screenshot when useful

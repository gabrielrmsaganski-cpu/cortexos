# Release Plan

## Short-term release target

Version theme:

- engine preserved
- product UI added
- demo mode and eval visibility added
- open source packaging improved

## Release checklist

- backend tests pass
- frontend build passes
- demo seed works
- UI served by FastAPI build
- docs updated
- README updated
- local validation recorded

## Recommended GitHub launch assets

- README
- architecture docs
- product spec
- verification log
- screenshots generated from demo mode
- short animated clip of Query Studio and Explain Center

## Honest positioning

Do not position CortexOS as:

- a universal autonomous agent platform
- a guaranteed long-term truth engine
- a replacement for application-specific data models

Position it as:

- a memory platform for agents
- a system that keeps verbatim memory as ground truth
- a lifecycle-aware retrieval and explainability layer
- a practical local-first open source system
